Invisible Internet Project

What is I2P?

I2P is a peer-to-peer Anonymous Network that uses

garlic routing
Garlic routing sends multiple encrypted messages (called "garlic cloves") together in one packet. This makes it harder to trace, as multiple messages are combined, and the network cannot easily distinguish which message belongs to which user.
(not to be confused with onion routing) to anonymize its users.
